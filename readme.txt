Simple yearly archive posts for any post type

== Description ==

A simple plugin to show yearly archive posts in frontend

== Installation ==

1. Upload plugin folder `dnt-simple-yearly-archive-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add new custom shortcode & settings through the "Archive Posts Shortcode" menu in WordPress
4. Place shortcode in your page content or anywhere you want to show it
